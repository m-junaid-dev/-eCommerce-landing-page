# Ecommerce Landing Page

This project is a simple eCommerce landing page built using HTML, CSS, and JavaScript.

## Project Purpose
I created this project to practice frontend web development and improve my UI design and JavaScript basics.

##  Features
- Responsive landing page layout
- Product showcase section
- Clean user interface
- Basic JavaScript interactions
- Modern styling with CSS

## Technologies Used
- HTML5
- CSS3
- JavaScript

##  Learning Outcome
Through this project, I improved my understanding of:
- Page layout structure
- Styling with CSS
- DOM interaction with JavaScript
- Creating real-world style landing pages

##  How to Run
Just open the `index.html` file in your browser.
